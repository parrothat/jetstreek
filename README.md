# Jetstreek

Jetstreek is a lightweight and open-source meteorological forecast map viewer 

It provides clean visualizations of weather parameters such as CAPE, temperature, wind, humidity, and precipitation using generated map layers.

## Features


- Forecast maps (CAPE, temperature, wind, humidity, precipitation)
- Simple and clean interface
- Works in any modern web browser

## What is Jetstreek?

Jetstreek is designed as a simple alternative to complex meteorological tools.  

It is ideal for:
- Quick weather overview
- Storm potential analysis
- Educational purposes

## Data & Maps

Jetstreek currently supports:

- CAPE (Convective Available Potential Energy)
- CIN (Convective Inhibition)
- Temperature
- Wind
- Humidity
- Precipitation

Planned features:
- Air pressure maps
- Radar imagery
- Expanded regional support (including USA)
